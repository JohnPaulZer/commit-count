Token Tutorial

How to get your classic GitHub token

  1. Open this page: https://github.com/settings/tokens/new
  2. Click Generate new token (classic).
  3. Type any token name and choose an expiration date.
  4. In Scopes, click repo.
  5. Click Generate token, copy it, and paste it into the field above.
